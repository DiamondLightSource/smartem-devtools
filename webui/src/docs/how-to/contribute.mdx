# Contributing

Thank you for your interest in contributing to SmartEM!

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Create a feature branch
4. Make your changes
5. Submit a pull request

## Code Standards

- Use British English in documentation
- No emojis in code or docs (Windows build compatibility)
- Follow existing patterns and conventions
- Add tests for new functionality

## Documentation

- Follow the [Diataxis](https://diataxis.fr/) framework
- Use ADRs for significant architectural decisions

## Questions?

Open an issue on GitHub for questions or suggestions.
